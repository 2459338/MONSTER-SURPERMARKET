概要

本ゲームは、電卓を使って4桁の引き算に挑戦する算数ゲームです。
かわいらしいモンスターたちが登場し、
楽しみながら計算力を鍛えられることを目的としています。

単なる計算問題ではなく、
「お店でのお買い物」というシチュエーションを通して、
実用的な引き算を体験できます。

ゲーム内容
■ モンスターのお買い物

ゲーム中、ランダムでモンスターたちがお店にやってきます。
モンスターはお客さんとしてお金を出し、
同時にお買い上げ金額が表示されます。

プレイヤーは、

支払ったお金 − お買い上げ金額

を計算し、お釣りの金額を答えます。

■ 計算ルール

使用する計算は 4桁の引き算

電卓を使って計算可能

正しいお釣りを導き出すことが目標

実際のお買い物に近い形式のため、
算数が苦手な人でも取り組みやすい設計になっています。

■ ポイントシステム

正解した場合：
→ お釣りの金額がポイントとして加算

不正解の場合：
→ ポイントは獲得できず、
→ 現在のポイントから300点減少

正確さが重要になるため、
慎重に計算することが求められます。

■ ゲームクリア条件

ポイントが 10,000点 に到達するとゲームクリアです。

かわいいモンスターたちと一緒に遊びながら、
自然と計算力が身につくゲームになっています。
